## 架构

### 目录结构

```
src/ralph_gui/
├── main.py          # 应用程序入口
├── app.py           # RalphApp 应用程序类
├── main_window.py   # 主窗口（视图层）
├── presenters/      # 表现层
├── services/        # 服务层（CLI、状态、配置）
├── models/          # 模型层
├── views/           # UI 组件
├── lib/             # Python 库工具
├── scripts/         # Python 脚本
└── templates/       # 模板文件
```

### MVP 模式

- **视图层**: PySide6 控件，仅处理 UI
- **表现层**: 业务逻辑，连接视图层与模型层
- **模型层**: 数据结构和状态
- **服务层**: 外部接口（CLI、文件 I/O）
