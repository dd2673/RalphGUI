## 项目概述

Ralph GUI 是一个 Windows 桌面应用程序，用于管理 Ralph for Claude Code 项目。采用 PySide6 (Qt) 构建，MVP 架构。

## 技术栈

- Python 3.10+
- PySide6 >= 6.6.0 (Qt GUI)
- PyYAML >= 6.0
- PyInstaller (打包)

## 目录结构

```
src/ralph_gui/
├── main.py           # 应用入口
├── app.py            # RalphApp 应用类
├── main_window.py    # 主窗口 (View)
├── presenters/       # Presenter 层 - 业务逻辑
├── services/         # Service 层 - CLI、状态、配置、日志
├── models/           # Model 层 - 数据模型
├── views/            # View 层 - UI 组件
├── lib/              # lib 层 - Python 库模块
├── scripts/          # scripts 层 - Python 脚本
├── templates/        # 模板文件
├── examples/         # 示例项目
└── i18n/            # 国际化
```

## MVP 架构

- **Views**: main_window.py, views/ - UI 组件，PySide6 widgets
- **Presenters**: presenters/ - 连接 View 与 Model，处理业务逻辑
- **Services**: services/ - CLI 调用、状态管理、配置管理、日志
- **Models**: models/ - 数据模型 (CircuitBreaker, LoopState, Project 等)

## 核心模型

- `CircuitBreakerModel` - 回路断路器状态 (CLOSED/HALF_OPEN/OPEN)
- `LoopStateModel` - 循环状态
- `Project` - 项目信息
- `RateLimitModel` - API 速率限制
- `SessionModel` - 会话管理

## 关键服务

- `CLIService` - 纯 Python CLI 服务
- `StateService` - 管理 Ralph 状态文件
- `ConfigService` - 管理 `.ralphrc` 配置
- `LogService` - 日志管理

## lib 层模块

- `circuit_breaker.py` - 断路器逻辑
- `date_utils.py` - 日期工具
- `enable_core.py` - 项目初始化核心
- `file_protection.py` - 文件保护
- `log_utils.py` - 日志工具
- `response_analyzer.py` - 响应分析
- `task_sources.py` - 任务源管理
- `timeout_utils.py` - 超时工具
- `wizard_utils.py` - 向导工具

## scripts 层

- `ralph_import.py` - PRD 导入
- `ralph_install.py` - 安装脚本
- `ralph_migrate.py` - 迁移脚本
- `ralph_monitor.py` - 监控脚本
- `ralph_setup.py` - 项目设置
- `ralph_stats.py` - 统计脚本
- `ralph_uninstall.py` - 卸载脚本
