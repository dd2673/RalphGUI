# 快速入门

本教程引导您快速上手 Ralph GUI，运行您的第一个自主开发循环。

## 前置要求

- Windows 10/11 (64-bit)
- Python 3.10+ (如从源码运行)
- Claude Code CLI (`npm install -g @anthropic-ai/claude-code`)

## 步骤 1：安装 Ralph GUI

### 方式一：使用打包版本

1. 下载最新版本的 `RalphGUI.exe`
2. 双击运行

### 方式二：从源码运行

```bash
# 克隆仓库
git clone https://github.com/your-repo/ralph-gui.git
cd ralph-gui

# 安装依赖
pip install -r requirements.txt

# 运行应用
python src/ralph_gui/main.py
```

## 步骤 3：选择项目目录

1. 点击「选择目录」按钮
2. 在弹出的对话框中选择包含 `.ralphrc` 文件的项目文件夹
3. 如果项目未初始化，应用会提示进行初始化

## 步骤 4：启动循环

1. 确保已选择有效的项目目录
2. 点击「启动」按钮
3. 应用将开始调用 Claude Code CLI 并执行循环任务
4. 日志区域会实时显示执行日志

## 界面功能说明

### 状态显示

| 组件 | 说明 |
|------|------|
| 断路器状态 | 显示 CLOSED/HALF_OPEN/OPEN |
| 循环状态 | 显示 已停止/运行中/已暂停 |
| 调用计数 | 显示 当前调用数 / 最大调用数 |

### 控制按钮

| 按钮 | 功能 |
|------|------|
| 启动 | 开始执行循环任务 |
| 暂停 | 暂停当前运行的循环 |
| 恢复 | 继续被暂停的循环 |
| 停止 | 终止当前循环任务 |

## 步骤 5：监控执行

执行过程中，您可以：

- **查看日志** - 实时滚动查看 Claude 输出
- **查看状态** - 监控断路器和调用计数
- **暂停/恢复** - 根据需要控制执行
- **停止** - 终止任务

## 常见问题

### 启动时提示"找不到 Claude Code"

确保已安装 Claude Code CLI：
```bash
npm install -g @anthropic-ai/claude-code
```

### 日志显示"断路器已打开"

应用自动检测到连续失败或无进展，已暂停调用。按「恢复」可重置断路器。

### 如何查看详细日志？

点击日志输出区域，使用滚动条查看完整日志。
