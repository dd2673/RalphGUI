# Ralph GUI 用户指南

本指南帮助您开始使用 Ralph GUI 并了解如何有效地管理您的项目。

## 指南

### [快速入门](01-快速入门.md)
一个实践教程，引导您快速上手 Ralph GUI。涵盖安装、项目选择和基本操作。

### [理解 Ralph 文件](02-理解Ralph文件.md)
了解 Ralph 项目创建哪些文件，哪些文件您应该自定义，以及它们如何协同工作。

### [编写有效需求](03-编写有效需求.md)
编写 PROMPT.md 的最佳实践，以及 fix_plan.md 在开发过程中如何演变。

## 主要功能

| 功能 | 说明 |
|------|------|
| 图形化界面 | 选择项目目录，启动/停止/暂停循环 |
| 断路器状态 | 实时显示回路断路器状态 (CLOSED/HALF_OPEN/OPEN) |
| 实时日志 | 监控面板显示执行日志 |
| 项目设置 | 配置管理和初始化 |
| 会话管理 | 支持会话连续性和过期控制 |
| 速率限制 | 内置 API 调用管理 |

## 快速参考

| 操作 | 方法 |
|-------------|---------|
| 启动应用 | `python src/ralph_gui/main.py` |
| 选择项目 | 点击「选择目录」按钮 |
| 启动循环 | 点击「启动」按钮 |
| 暂停循环 | 点击「暂停」按钮 |
| 停止循环 | 点击「停止」按钮 |
| 查看设置 | 点击「设置」按钮 |

## 状态说明

### 断路器状态

| 状态 | 说明 | 操作建议 |
|------|------|----------|
| CLOSED | 正常运行 | 无需操作 |
| HALF_OPEN | 部分恢复 | 观察是否正常工作 |
| OPEN | 已触发保护 | 检查日志，按「恢复」重置 |

### 循环状态

| 状态 | 说明 |
|------|------|
| 已停止 | 循环未运行 |
| 运行中 | 正在执行任务 |
| 已暂停 | 循环被用户暂停 |
