# Ralph 用户指南

本指南帮助您开始使用 Ralph 并了解如何有效地为您的项目配置它。

## 指南

### [快速开始：您的第一个 Ralph 项目](01-快速入门.md)
一个实践教程，引导您在现有项目上启用 Ralph 并运行您的第一个 autonomous 开发循环。您将从零开始构建一个简单的 CLI todo 应用程序。

### [理解 Ralph 文件](02-理解Ralph文件.md)
了解 Ralph 创建哪些文件，哪些文件您应该自定义，以及它们如何协同工作。包括完整的参考表和文件关系说明。

### [编写有效需求](03-编写有效需求.md)
编写 PROMPT.md 的最佳实践，何时使用 specs/，以及 fix_plan.md 在开发过程中如何演变。包括好的和坏的示例。

## 快速参考

| 我想要... | 这样做 |
|-------------|---------|
| 在现有项目上启用 Ralph | `ralph-enable` |
| 导入 PRD/需求文档 | `ralph-import requirements.md project-name` |
| 从零开始创建新项目 | `ralph-setup my-project` |
| 使用监控启动 Ralph | `ralph --monitor` |
| 检查 Ralph 正在做什么 | `ralph --status` |
