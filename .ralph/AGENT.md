# Ralph Agent 配置指南

## 项目概述

Ralph GUI 是一个基于 PySide6 的 Windows 桌面应用程序，用于管理 Ralph for Claude Code 项目的循环执行。核心能力是**循环-回调-检测刷新状态-重启下一轮循环-持续迭代**。

## 核心能力约束（严禁修改）

```
┌───────────────────────────────────────────────────────────────┐
│  启动(Start) → 执行(Do) → 检测刷新状态(Check) → 回调(Callback)  │
│       ↑                                               │       │
│       └──────────────── 重启(Next) ←──────────────────┘       │
└───────────────────────────────────────────────────────────────┘
```

**绝对禁止修改的核心组件**:
- `LoopPresenter._run_loop()` - 主循环结构
- `CircuitBreakerModel` - 断路器状态机
- `CLIService.start_loop()` - 回调机制
- `LoopOptions` - 配置项语义

## 代码审查清单

修改代码前必须检查:

- [ ] 是否修改了核心循环结构？
- [ ] 是否添加了新的状态转换？
- [ ] 是否改变了配置项的语义？

- [ ] 是否直接兼容中文？

## 注意事项

1. **不要破坏核心循环**: `_run_loop()` 是唯一的主循环入口
2. **不要跳过断路器**: 断路器是死循环防护的核心
3. **不要忽略异常**: 所有 `except` 必须记录日志
4. **不要泄漏资源**: 进程和线程必须正确清理
